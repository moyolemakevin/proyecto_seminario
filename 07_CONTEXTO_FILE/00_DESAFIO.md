# Desafio

Construir un sistema de alerta temprana para pedidos de OLIST con riesgo de
entrega tardia.

El sistema debe usar datos historicos de pedidos para responder:

Que tan probable es que un pedido nuevo llegue tarde?

La salida esperada es una probabilidad y un nivel de riesgo:

bajo
medio
alto

El resultado se usa para priorizar seguimiento logistico y anticipar problemas
de experiencia del cliente.
